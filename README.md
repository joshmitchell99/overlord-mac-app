# Overlord for Mac

Download the latest release: **[Overlord.dmg](https://github.com/joshmitchell99/overlord-mac-app/releases/latest/download/Overlord.dmg)**

Overlord is a Mac productivity app that blocks distracting apps and websites, runs AI check-ins when you drift, and helps you stay focused.

## Installation

1. Download `Overlord.dmg` from the link above
2. Open the DMG and drag Overlord to Applications
3. Launch Overlord from Applications or Spotlight

## Auto-updates

Once installed, Overlord checks for updates automatically and installs them in the background. You can also manually check via Settings → Check for updates.
